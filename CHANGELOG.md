# Changelog

## [1.0.0] - 2026-05-25
### Added
- Discrete Event Simulation (DES) engine with Event, Timeout, Process, Environment
- 6 CPU scheduling algorithms: FCFS, SJF, SRTF, Priority, Priority (Preemptive), Round Robin
- Memory Manager with contiguous allocation
- Virtual File System (VFS) with hierarchical directories, files, and disk quotas
- I/O Device Manager with blocking queues and latency simulation
- Tkinter + Matplotlib real-time GUI dashboard
- CLI with simulation mode, VFS shell, JSON config loading, CSV/JSON export
- Benchmark suite for scheduler comparison
- 31 unit tests covering engine, scheduler, memory, VFS, I/O, config, exporter
- GitHub Actions CI workflow
- pyproject.toml, Makefile, .gitignore
